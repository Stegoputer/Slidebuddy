SLIDE-GENERIERUNG (BATCH):

Du generierst MEHRERE Folien auf einmal fuer ein Kapitel.

REGELN:
1. Generiere IN DER GEGEBENEN REIHENFOLGE
2. Roter Faden zwischen Folien — jede baut auf der vorherigen auf
3. Keine inhaltlichen Wiederholungen zwischen Folien
4. Nutze RAG-Quellen als Faktengrundlage wo passend
5. Halte dich EXAKT an die JSON-Struktur des zugewiesenen Templates
6. Sprechernotizen: ausformulierter Redetext, KEINE Begruessung/Willkommen, direkt starten
7. Generiere EXAKT so viele Folien wie angegeben

STIL:
- Vollstaendige Saetze, keine reinen Stichworte
- Konkrete Zahlen/Fakten statt vager Aussagen
- Variation in Satzstruktur und Einstiegssaetzen

TEXTUMFANG — halte dich STRIKT an die angegebene Stufe:
- "Kurz": Folientext max 3-4 Zeilen. Sprechernotizen 2-3 Saetze (~50 Woerter).
- "Mittel": Folientext max 5-6 Zeilen. Sprechernotizen 4-6 Saetze (~100 Woerter).
- "Ausfuehrlich": Folientext max 7-8 Zeilen. Sprechernotizen 7-10 Saetze (~180 Woerter).
Sprechernotizen: natuerlicher Redefluss, Uebergaenge, direkt starten ohne Begruessung.

AUSGABEFORMAT — NUR dieses JSON, kein Text davor/danach:

{
  "slides": [
    {
      "slide_index": 1,
      "template_type": "...",
      "title": "Folientitel",
      "subtitle": "Untertitel oder null",
      "content": { ... template-spezifische Felder ... },
      "speaker_notes": "Ausformulierter Redetext...",
      "chain_of_thought": "Denkprozess...",
      "key_summary": "Kernaussage in einem Satz"
    }
  ]
}

WICHTIG:
- Valides JSON, alle Strings in doppelten Anfuehrungszeichen
- "content" muss EXAKT die Felder des Templates enthalten, keine eigenen erfinden
- "subtitle" darf null sein
